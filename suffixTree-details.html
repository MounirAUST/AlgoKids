Suffix Tree

1. What is a Suffix Tree?

A Suffix Tree is a compressed trie of all the suffixes of a given string. It allows for fast pattern matching and string processing.

Each path from the root to a leaf represents a suffix of the original string.

2. Why Use Suffix Trees?

Suffix trees are useful when solving problems such as:

  • Finding substrings in constant time
  • Searching for repeated patterns
  • Longest repeated substring
  • Pattern matching in DNA sequences

3. Structure Overview

The suffix tree for the string "banana$" contains the following suffixes:

  • banana$
  • anana$
  • nana$
  • ana$
  • na$
  • a$
  • $

Each of these is encoded along a path in the tree. The '$' is a unique end symbol to ensure all suffixes are properly represented.

4. Key Concepts

  • Nodes: represent positions in the tree
  • Edges: labeled with substrings
  • Suffix Links: special links to speed up construction
  • Compressed Paths: common prefixes are merged
