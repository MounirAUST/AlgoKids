Union-Find (Disjoint Set)

1. What is Union-Find?

Union-Find (or Disjoint Set Union, DSU) is a data structure that keeps track of elements partitioned into non-overlapping sets. It's especially useful for checking connectivity and merging groups.

2. Why Use Union-Find?

Union-Find is used in many algorithms that require grouping or connectivity checks. Examples include:

  • Detecting cycles in a graph
  • Building a Minimum Spanning Tree (Kruskalâ€™s algorithm)
  • Grouping related items

3. Core Operations

  • Find(x): Determines the root of the set containing x
  • Union(x, y): Merges the sets containing x and y

With optimizations like path compression and union by rank, these operations become extremely efficient.

4. Key Concepts

  • Set Representative: The root node that identifies a set
  • Path Compression: Flattens the tree to make future finds faster
  • Union by Rank: Attaches the smaller tree under the root of the larger tree

6. Summary

The Union-Find structure helps solve complex grouping problems efficiently. It's a vital tool in graph theory, especially for maintaining connected components and optimizing dynamic grouping operations.
