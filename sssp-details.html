Single-Source Shortest Paths

1. What is SSSP?

Single-Source Shortest Paths (SSSP) is the problem of finding the shortest path from one source node to all other nodes in a weighted graph.

This is useful for navigation, routing, and solving optimization problems in graphs.

2. Why It’s Important

SSSP algorithms are used in:

  • GPS and navigation systems
  • Network routing
  • Game pathfinding
  • Project scheduling and more

3. Popular Algorithms

  • Dijkstra’s Algorithm: Works with non-negative edge weights
  • Bellman-Ford Algorithm: Handles negative edge weights
  • BFS: Used for unweighted graphs to find the shortest number of edges

4. Key Concepts

  • Priority queues for efficient distance updates
  • Relaxation: improving the shortest path estimate step-by-step
  • Edge weights influence the cost of each path
