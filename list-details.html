Linked List, Stack & Queue

1. What is a Linked List?

A Linked List is a data structure made of nodes, where each node contains a value and a pointer to the next node. It allows efficient insertions and deletions.

  • Can grow or shrink in size dynamically
  • Useful when the number of elements is unknown ahead of time
  • Can be singly linked or doubly linked

2. What is a Stack?

A Stack is a linear data structure that follows the Last In, First Out (LIFO) principle. The last item added is the first to be removed.

  • push(x): Add element to the top
  • pop(): Remove the top element
  • peek(): View the top element without removing

Think of it like a stack of plates — you remove the top one first!

3. What is a Queue?

A Queue is a linear data structure that follows the First In, First Out (FIFO) principle. The first item added is the first to be removed.

  • enqueue(x): Add element to the back
  • dequeue(): Remove the front element
  • peek(): View the front element without removing

Think of it like a line at a ticket counter — first come, first served.

5. Summary

These structures are the building blocks of more advanced algorithms. Mastering them helps you understand how programs manage memory, store data, and control process flow efficiently.
